Ejercicios en JavaScript - Área de Medio Ambiente

1. Clasificación de calidad del aire (AQI)

Ingresar el valor AQI y clasificar: bueno, moderado, dañino, etc.

2. Registrar niveles de ruido ambiental

Calcula el promedio de un set de datos de ejemplo (niveles en dB).

3. Contar focos de calor

Cuenta cuántas temperaturas en un set de datos superan los 45°C (se detiene en 0).

4. Clasificación del tipo de residuo

Según código 1-4: orgánico, plástico, papel/cartón, vidrio.

5. Monitoreo de niveles del río

Simula el registro de niveles de un río hasta que se decide parar.
